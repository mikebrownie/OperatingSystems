Man Page for Multi-Lookup

NAME

multi-lookup - resolve a set of hostnames to IP addresses

SYNOPSIS

./multi-lookup <# requester> <# resolver> <requester log> <resolver log> [ <data file> ... ]

DESCRIPTION

The file names specified by are passed to the pool of requester threads which place information into a shared data area. Resolver threads read the shared data area and find the corresponding IP address.

PARAMETERS

<# requesters> number of requestor threads to place into the thread pool.

<# resolvers> number of resolver threads to place into the thread pool.

name of the file into which all the requester status information is written.

name of the file into which all the resolver status information is written.

filename to be processed. Each file contains a list of host names, one per line, that are to be resolved.
